机器人的运动范围

这题我自己用的是DFS，开始没想起来BFS怎么做的，想的是用递归做BFS，看了题解老哥BFS恍然大悟，复习时重写一定用BFS写，下面是DFS代码

class Solution {
    public int movingCount(int m, int n, int k) {
        if(k == 0)
            return 1;
        boolean[][] im = new boolean[m][n];
        return dfs(m,n,0,0,k,im);
    }

    public int dfs(int m, int n, int i, int j, int k, boolean[][] im) {
        if(i>=m || j>=n || im[i][j] || calu(i,j)>k)
            return 0;
        im[i][j] = true;
        return 1 + dfs(m,n,i+1,j,k,im) + dfs(m,n,i,j+1,k,im);
    }

    public int calu(int i, int j) {
        int x=0;
        while(i>0) {
            x+=i%10;
            i=i/10;
        }
        while(j>0) {
            x+=j%10;
            j=j/10;
        }
        return x;
    }
}

最长连续序列

官方题解有Hashset做的，我这个用Hashmap做的又复杂又慢，几十倍的时间消耗，复习重写要用官解写法，另外这是第一道没看题解一边通过的困难级题目，mark一下

class Solution {
    HashMap<Integer,Integer> map;

    public int longestConsecutive(int[] nums) {
        if(nums.length == 0)
            return 0;
        map = new HashMap();
        int max = 1;
        for(int i = 0;i < nums.length;i++) {
            if(map.containsKey(nums[i]))
                continue;
            map.put(nums[i],1);
            update(nums[i]);
            if(max<map.get(nums[i]))
                max = map.get(nums[i]);
        }
        return max;
    }

    public void update(int num) {
        int k = num;
        int len = 0;
        if(map.containsKey(num-1) && map.containsKey(num+1)) {
            len = map.get(num-1)+1+map.get(num+1);
            while(map.containsKey(k)) {
                map.put(k,len);
                k--;
            }
            k = num;
            while(map.containsKey(k)) {
                map.put(k,len);
                k++;
            }
        }
        if(map.containsKey(num-1)) {
            len = map.get(num-1)+1;
            while(map.containsKey(k)) {
                map.put(k,len);
                k--;
            }
        }
        if(map.containsKey(num+1)) {
            len = map.get(num+1)+1;
            while(map.containsKey(k)) {
                map.put(k,len);
                k++;
            }
        }
    }
}

把数字翻译成字符串

自己是用最普通的动态规划做的，事实上看了题解才想到可以类似与斐波那契数列那样不用数组，空间压缩只使用两个变量交替前行。另外也可以将每次前进分而治之，使用递归求解。

class Solution {
    public int translateNum(int num) {
        if(num/10 == 0)
            return 1;
        String res = Integer.toString(num);
        int[] dp = new int[res.length()];
        dp[0] = 1;
        if(cal(res.charAt(0),res.charAt(1)) <= 25)
            dp[1] = 2;
        else
            dp[1] = 1;
        for(int i=2;i<res.length();i++) {
            if((cal(res.charAt(i-1),res.charAt(i)) <= 25) &&
                (cal(res.charAt(i-1),res.charAt(i)) >= 10))
                dp[i] = dp[i-1] + dp[i-2];
            else
                dp[i] = dp[i-1];
        }
        return dp[res.length()-1];
    }

    public int cal(char a,char b) {
        int num1 = a-'0';
        int num2 = b-'0';
        return num1*10+num2;
    }
}

class Solution {
    public int cuttingRope(int n) {
        if(n<=3) return n-1;
        int b=n%3,p=1000000007;
        long rem=1,x=3;
        for(int a=n/3-1;a>0;a/=2) {
            if(a%2==1)
                rem = (rem*x)%p;
            x = (x*x)%p;
        }
        if(b == 0) return (int)(rem * 3 % p);
        if(b == 1) return (int)(rem * 4 % p);
        return (int)(rem * 6 % p);
    }
}

数值的整数次方

与上一题一样思考如何快速求幂，发现自己对二进制的位操作以及类型的边界值不是很了解，另外自己的代码写的太冗余了

class Solution {
    public double myPow(double x, int n) {
        if(n==0)
            return 1.0;
        double res=1.0;
        if(n<0) {
            long b=n;
            b = -b;
            x = 1/x;
            while(b>0) {
                if(b%2 == 0) {
                    x *= x;
                    b /= 2;
                }
                if(b%2 == 1) {
                    res*=x;
                    x *= x;
                    b=(b-1)/2;
                }
            }
        }
        while(n>0) {
            if(n%2 == 0) {
                x *= x;
                n /= 2;
            }
            if(n%2 == 1) {
                res*=x;
                x *= x;
                n=(n-1)/2;
            }
        }
        return res;
    }
}

二进制中1的个数

上一题提到的对二进制的位操作与十进制的联系很不熟悉，导致自己的代码看起来很蠢。

贴一个之前那个天秀老哥的题解链接：Krahets

public class Solution {
    // you need to treat n as an unsigned value
    public int hammingWeight(int n) {
        String result = Integer.toBinaryString(n);
        int num = 0;
        for(int i=0;i<result.length();i++) {
            if(result.charAt(i) == '1')
                num++;
        }
        return num;
    }
}
