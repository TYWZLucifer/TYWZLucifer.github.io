打家劫舍1

与官方题解不同，我将dp数组中的每一个数定义为如果一定要偷这家所能得到的最大金额，这样从第四家开始之后的每一家，例如第n家，dp[n]只与n-2、n-3位置保存的金额数有关，用一个max变量记录最大金额，最后返回即可。

class Solution {
    public int rob(int[] nums) {
        int max = 0;
        if(nums.length <= 2) {
            for(int i=0;i<nums.length;i++) {
                max = Math.max(max,nums[i]);
            }
            return max;
        }
        int[] dp = new int[nums.length];
        dp[0] = nums[0];
        dp[1] = nums[1];
        dp[2] = nums[2] + dp[0];
        max = Math.max(dp[0],dp[1]);
        max = Math.max(max, dp[2]);
        for(int i=3;i<nums.length;i++) {
            dp[i] = Math.max(dp[i-3]+nums[i],dp[i-2]+nums[i]);
            max = Math.max(max,dp[i]);
        }
        return max;
    }
}

青蛙跳台阶

这个题其实和斐波那契数列很像，用上面一题中动态规划的思路就可以解决，重点是找好数组保存的是什么状态，状态如何转移，想明白这两个问题就很容易写。

我个人的方法是使用递归，但会发现空间占用太大，优化策略是使用hashmap保存每一步计算得到的”新”的状态，这样每次需要计算时先去里面找找看有没有，如果有就不需要重复递归了。

class Solution {
    HashMap<Integer,Integer> map;
    public Solution() {
        map = new HashMap();
    }
    public int numWays(int n) {
        if(n==0 || n==1)
            return 1;
        if(map.containsKey(n)) {
            return map.get(n);
        }
        else {
            map.put(n,(numWays(n-1)+numWays(n-2))%1000000007);
        }
        return map.get(n);
    }
}

二维数组的查找

这个题其实就是一棵树上的查找，想明白root节点在哪很容易做

class Solution {
    public boolean findNumberIn2DArray(int[][] matrix, int target) {
        if(matrix.length == 0)
            return false;
        int i=0,j=0;
        j = matrix[0].length - 1;
        while(i<matrix.length && j >= 0) {
            if(matrix[i][j] < target) {
                i++;
                continue;
            }
            if(matrix[i][j] > target) {
                j--;
                continue;
            }
            if(matrix[i][j] == target)
                return true;
        }
        return false;
    }
}

顺时针打印矩阵

直接按照元素的总个数循环遍历，或者递归，但最重要的是考虑如何进行方向变化的判定与操作。下面使用递归写法：

class Solution {
    private int[][] direction = {{0,1},{1,0},{0,-1},{-1,0}};

    public int[] spiralOrder(int[][] matrix) {
        if(matrix.length == 0)
            return new int[0];
        int[] out = new int[matrix.length*matrix[0].length];
        int[][] im = new int[matrix.length][matrix[0].length];
        find(matrix,out,0,0,0,0,im);
        return out;
    }

    public boolean find(int[][] matrix, int[] out, int k, int i, int j, int di, int[][] im) {
        if(i<0 || i>=matrix.length || j<0 || j>=matrix[0].length || im[i][j]<0)
            return false;
        if(k == out.length)
            return true;
        out[k] = matrix[i][j];
        im[i][j] = -1;
        k++;

        int dx = i + direction[di][0];
        int dy = j + direction[di][1];
        if(find(matrix,out,k,dx,dy,di,im))
            return true;
        
        di = (di+1)%4;
        dx = i + direction[di][0];
        dy = j + direction[di][1];
        return find(matrix,out,k,dx,dy,di,im);
    }
}

旋转数组的最小数字

其实从后往前遍历不是不可以，只是时间复杂度较高，看起来不秀，思考使用二分法降低时间复杂度

class Solution {
    public int minArray(int[] numbers) {
        // for(int i=numbers.length-1;i >= 1;i--) {
        //     if(numbers[i] < numbers[i-1])
        //         return numbers[i];
        // }
        // return numbers[0];//遍历的写法

        int i=0,j=numbers.length-1,m=(i+j)/2;
        while(i!=j) {
            if(numbers[m] < numbers[j])
                j = m;
            else if(numbers[m] > numbers[j]) 
                i = m+1;
            else
                j = j-1;
            m = (i+j)/2;
        }
        return numbers[j];
    }
}

重建二叉树

大二其实学过的，有了前序遍历和中序遍历，找对root，分割好数组就可以了，官方题解的迭代法好复杂我不想看QAQ…..

/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode(int x) { val = x; }
 * }
 */
class Solution {
    public TreeNode buildTree(int[] preorder, int[] inorder) {
        List<Integer> pre = new ArrayList<Integer>();
        for(int num : preorder)
            pre.add(num);
        List<Integer> in = new ArrayList<Integer>();
        for(int num : inorder)
            in.add(num);
        return F(pre,in);
    }

    public TreeNode F(List pre, List in) {
        if(pre.size() == 0)
            return null;;
        TreeNode root = new TreeNode((int)pre.get(0));
        int index = in.indexOf(root.val);
        root.left = F(pre.subList(1,index+1),in.subList(0,index));
        root.right = F(pre.subList(index+1,pre.size()),in.subList(index+1,in.size()));
        return root;
    }
}

矩阵中的路径

从矩阵中找路径，能回溯原来的状态很关键，看了别人的题解，DFS，剪枝，最好自己多敲两边

class Solution {
    public boolean exist(char[][] board, String word) {
        char[] words = word.toCharArray();
        for(int i=0;i<board.length;i++) {
            for(int j=0;j<board[0].length;j++) {
                if(dfs(board,words,i,j,0)) return true;
            }
        }
        return false;
    }

    public boolean dfs(char[][] board,char[] words,int i,int j,int k) {
        if(i<0 || i>=board.length || j<0 || j>=board[0].length || board[i][j]!=words[k])
            return false;
        if(k == words.length-1)
            return true;
        char tmp = board[i][j];
        board[i][j] = '/';
        boolean res = dfs(board,words,i+1,j,k+1) || dfs(board,words,i-1,j,k+1) ||
              dfs(board,words,i,j+1,k+1) || dfs(board,words,i,j-1,k+1);
        board[i][j] = tmp;
        return res;
    }
}
