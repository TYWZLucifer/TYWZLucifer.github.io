地下城游戏

从开始往前走需要同时考虑最低初始健康和过程中的路径和两个值，所以“有两个重要程度相同的参数同时影响后续的决策，这样的动态规划是不满足「无后效性」的”

class Solution {
    public int calculateMinimumHP(int[][] dungeon) {
        int n=dungeon.length, m=dungeon[0].length;
        int[] dp = new int[m+1];
        Arrays.fill(dp,Integer.MAX_VALUE);
        dp[m-1]=1;
        for(int i=n-1;i>=0;i--) {
            for(int j=m-1;j>=0;j--) {
                dp[j] = Math.max(1,Math.min(dp[j],dp[j+1])-dungeon[i][j]);
            }
        }
        return dp[0];
    }
}

两个数组的交集2

很容易想到用hashmap做，但巧妙在于一定去更短的那个数组放到map里

class Solution {
    public int[] intersect(int[] nums1, int[] nums2) {
        if (nums1.length > nums2.length) {
            return intersect(nums2, nums1);
        }
        HashMap<Integer,Integer> map = new HashMap();
        for(int num:nums1) {
            int count = map.getOrDefault(num, 0) + 1;
            map.put(num, count);
        }

        int[] intersection = new int[nums1.length];
        int index = 0;

        for(int num:nums2) {
            int count = map.getOrDefault(num,0);
            if(count > 0) {
                intersection[index++] = num;
                count--;
                if (count > 0) map.put(num, count);
                else map.remove(num);
            }
        } 

        return Arrays.copyOfRange(intersection, 0, index);
    }
}

和为s的两个数字

笨比才用两数之和的方法做，人家说了是递增排序肯定是有用的，用双指针滑动窗口就可以

class Solution {
    public int[] twoSum(int[] nums, int target) {
        HashMap<Integer,Integer> map = new HashMap();
        for(int num:nums) {
            if(map.containsKey(num)) return new int[]{num,target-num};
            map.put(target-num,num);
        }
        return new int[0];
    }
}

三角形最小路径和

题不难想到解法，但边界的转移需要注意

class Solution {
    public int minimumTotal(List<List<Integer>> triangle) {
        int n = triangle.size();
        int[] f = new int[n];
        f[0] = triangle.get(0).get(0);
        int min = Integer.MAX_VALUE;
        for(int i=1;i<n;i++) {
            f[i] = f[i-1]+triangle.get(i).get(i);
            for(int j=i-1;j>0;j--) f[j] = Math.min(f[j],f[j-1])+triangle.get(i).get(j);
            f[0] += triangle.get(i).get(0);
        }
        for(int i=0;i<n;i++) min = Math.min(min,f[i]);
        return min;
    }
}

class Solution {
    public int[][] findContinuousSequence(int target) {
        int i=1, j=1, sum=0;
        List<int[]> res = new ArrayList<>();
        while(i<=target/2) {
            if(sum < target) sum += j++;
            else if(sum > target) sum -= i++;
            else {
                int[] arr = new int[j-i];
                for(int k=i;k<j;k++) arr[k-i] = k;
                res.add(arr);
                sum -= i++;
            }
        }
        return res.toArray(new int[res.size()][]); // 这里的new int[][]没搞清楚
    }
}

不同的二叉搜索树

卡特兰数 是真滴牛批，灵魂画手的题解写的很清楚

class Solution {
    public int numTrees(int n) {
        int[] dp = new int[n+1];
        dp[0] = 1;
        dp[1] = 1;
        for(int i=2;i<=n;i++) {
            for(int j=1;j<=i;j++) dp[i] += dp[j-1]*dp[i-j];
        }
        return dp[n];
    }
}
